# Project Specification

## Overview

Build a modern web application with authentication and user management.

---

## Stack

- Backend: Spring Boot (REST API)
- Frontend: React (SPA)
- Styling: TailwindCSS

---

## Features

- Authentication (JWT)
- User CRUD

---

## Constraints

- Do NOT use server-side rendering
- Do NOT use Thymeleaf
- Do NOT mix frontend frameworks
