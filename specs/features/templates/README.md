# Specialized feature templates

Use these optional templates when a feature matches a common pattern:

- `crud/` - create/read/update/delete flows
- `auth-hardening/` - security and session improvements
- `pagination/` - paginated list behavior
- `bootstrap-ui/` - Bootstrap 5 UI refresh/implementation for SSR + HTMX screens

Default scaffolding still starts from `specs/features/TEMPLATE/`.
