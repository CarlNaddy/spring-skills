# Feature spec: pagination

## Summary

Add pagination for list-heavy screens to improve usability and performance.

## Scope

### In scope

- Page size and page index handling
- Next/previous navigation in UI
- Empty and boundary state behavior

### Out of scope

- Infinite scroll
- Advanced sorting/filtering redesign

## Acceptance criteria

- [ ] User can navigate between pages
- [ ] Page boundaries are handled safely
- [ ] Pagination state remains consistent after list updates
